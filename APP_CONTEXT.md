## Quick facts (see README for the full write-up)

- Nonbank is a hybrid DeFi wallet with a seamlessly integrated custodial Visa
  card (NON×CARD). Non-custodial wallet + custodial card (Kolo issuer).
- Card live in 100+ countries (EU, APAC, LATAM, Central Asia, Middle East;
  excludes TUR, ISR, CHN, IND, US).
- EN-global content strategy — no country matrix, no localization pillars.
- Differentiators: gasless fees, built-in AML Watchtower, hybrid DeFi + card,
  proxy addresses / watch wallets, NON ID.
- Real competitors: Gnosis Pay, MetaMask Card, COCA, Bleap (self-custody + card);
  Crypto.com, Bybit Card, Nexo Card, RedotPay, Holyheld, Moon (adjacent).
