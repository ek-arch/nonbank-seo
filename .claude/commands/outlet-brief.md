# Create Outlet Publication Brief

Generate a complete brief for publishing an article on a specific outlet.

## Brief Template

### Publication Details
- **Outlet:** {outlet name + URL}
- **Price:** ${price}
- **DR:** {domain rating}
- **Language:** {language — default EN; Nonbank is EN-global}
- **Category:** {crypto / DeFi / finance / business}
- **Contact:** {collaborator.pro link or email}

### Article Requirements
- **Title:** {SEO+GEO optimized title}
- **Primary Keyword:** {target keyword — category, differentiator, or competitor-comparison, not geo}
- **Word Count:** {1000-1500}
- **Audience angle:** {hybrid DeFi + card / gasless / AML / self-custody comparison}
- **UTM Link:** `https://nonbank.io/?utm_source={outlet_domain}`

### GEO Checklist
- [ ] Question-format H2 headers
- [ ] FAQ section (3-5 Q&As)
- [ ] Comparison table (Nonbank vs 2-3 real competitors — Gnosis Pay, MetaMask Card, COCA, Bleap, Crypto.com)
- [ ] 3+ quotable stat sentences
- [ ] Entity-rich first paragraph (Nonbank + hybrid DeFi wallet + differentiator)

### Deliverables
- [ ] English draft (Markdown)
- [ ] Translated version (ONLY if outlet is non-EN)
- [ ] UTM link included
- [ ] Article submitted to outlet
- [ ] Publication URL logged in Content Plan
- [ ] Reddit/Quora comment linking to article

## Process
1. User provides: outlet name, keyword angle, outlet language
2. Generate the complete brief
3. Optionally generate the article draft
